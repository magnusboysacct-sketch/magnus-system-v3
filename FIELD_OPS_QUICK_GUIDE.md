# Field Operations - Quick User Guide

## Getting Started

### Access Field Ops

**From Desktop/Tablet:**
1. Login to Magnus System
2. Click "Field Ops" in sidebar (smartphone icon)
3. Dashboard opens with today's activity

**From Mobile:**
1. Login to Magnus System
2. Tap menu (hamburger icon)
3. Tap "Field Ops"
4. Dashboard opens full-screen

## Daily Log Entry (30-60 seconds)

### Quick Steps

1. **Tap "Daily Log" (blue card)**
   - Form opens in modal
   - Today's date already selected

2. **Select Weather (optional, 1 tap)**
   - ☀️ Sunny
   - ☁️ Cloudy
   - 🌧️ Rainy
   - 💨 Windy
   - ❄️ Snowy

3. **Enter Worker Count (optional)**
   - Tap number field
   - Enter count
   - Number pad opens automatically

4. **Describe Work (optional)**
   - Tap "Work Performed"
   - Type or speak description
   - Field expands as you type

5. **Add Details (all optional)**
   - Deliveries received
   - Issues encountered
   - Additional notes

6. **Save**
   - Tap large "Save Log" button
   - Form closes automatically
   - Today's log appears on dashboard

### Edit Today's Log

If you already created today's log:
- Button shows "Update Today" instead of "Create New"
- Form opens with existing data
- Edit any fields
- Save to update

## Photo Upload (10-20 seconds)

### Camera Capture

1. **Tap "Add Photos" (green card)**
   - Modal opens with two options

2. **Tap "Take Photo"**
   - Camera opens (back camera)
   - Take photo
   - Returns to preview

3. **Add More Photos (optional)**
   - Tap "Add More"
   - Take another photo
   - Repeat as needed

4. **Add Captions (optional)**
   - Tap caption field under each photo
   - Type description
   - Leave blank if not needed

5. **Upload**
   - Tap "Upload X Photos"
   - Wait for upload
   - Form closes automatically
   - Photos appear on dashboard

### Gallery Upload

1. **Tap "Add Photos" (green card)**
2. **Tap "Choose Photos"**
   - Gallery opens
   - Select one or more photos
   - Tap "Done"
3. **Review Previews**
   - All photos show in list
   - Add captions if needed
   - Remove any unwanted (X button)
4. **Upload**
   - Tap "Upload X Photos"
   - Photos upload together

## Dashboard Features

### Today's Activity
- Real-time feed of today's updates
- Shows time of each activity
- Automatically filters to current day
- See team activity as it happens

### Today's Log Summary
- Shows if you created today's log
- Displays weather, workers, work preview
- Quick edit button
- Highlighted with blue border

### Recent Logs
- Last 5 daily logs
- Calendar-style date display
- Weather emoji shown
- Worker count badge
- Tap to view/edit

### Recent Photos
- Last 6 photos uploaded
- 3-column grid
- Square thumbnails
- Tap to view (coming soon)

## Tips & Best Practices

### Daily Logs

**Frequency**
- Create one log per day
- Update throughout the day as needed
- Complete by end of shift

**What to Include**
- Major work completed
- Materials delivered
- Any delays or issues
- Safety concerns
- Weather conditions affecting work

**What's Optional**
- Everything except the date!
- Add as much or little detail as needed
- Can always edit later

### Photos

**What to Photograph**
- Work in progress
- Completed work
- Material deliveries
- Site conditions
- Safety setups
- Issues or defects

**Best Practices**
- Take multiple angles
- Include context
- Capture before/after
- Document daily progress
- Add captions for clarity

**When to Upload**
- Throughout the day as work happens
- Before leaving site
- When documenting deliveries
- Immediately for issues

### Mobile Tips

**Battery Conservation**
- Close camera between shots
- Upload photos over WiFi when possible
- Reduce screen brightness

**Data Usage**
- Photos use cellular data if uploading
- Wait for WiFi for large batches
- Check data plan

**Quick Entry**
- Use voice-to-text for descriptions (device feature)
- Weather is one tap
- Worker count is quick number entry
- Skip optional fields

## Troubleshooting

### Can't Access Field Ops

**Issue:** Field Ops not in menu
**Solution:**
- Make sure you're logged in
- Verify you're a project member
- Contact admin if issue persists

**Issue:** No project selected
**Solution:**
- Return to dashboard
- Select a project from dropdown
- Return to Field Ops

### Daily Log Issues

**Issue:** Can't create log for today
**Solution:**
- Check if log already exists for today
- Button should say "Update Today"
- Edit existing log instead

**Issue:** Date won't select
**Solution:**
- Can't select future dates (by design)
- Today and past dates only
- Change device date if incorrect

**Issue:** Form won't submit
**Solution:**
- Verify date is selected
- Check internet connection
- Try again
- Check for error message

### Photo Issues

**Issue:** Camera won't open
**Solution:**
- Check browser camera permissions
- Allow camera access when prompted
- Try gallery upload instead
- Check device camera functionality

**Issue:** Photos won't upload
**Solution:**
- Check internet connection
- Verify file size not too large
- Try one photo at a time
- Check storage bucket permissions

**Issue:** Preview not showing
**Solution:**
- Wait for image to load
- Check file is image type
- Try different photo
- Refresh page

### Performance Issues

**Issue:** Slow loading
**Solution:**
- Check internet connection
- Close other apps
- Clear browser cache
- Try different time of day

**Issue:** Dashboard not updating
**Solution:**
- Refresh page
- Close and reopen Field Ops
- Check internet connection
- Log out and back in
