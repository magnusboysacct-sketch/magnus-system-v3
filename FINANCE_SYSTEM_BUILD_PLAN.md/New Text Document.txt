# 🧭 BUILD PHASES (STRICT ORDER)

## 🔹 PHASE 1 — ACCOUNTING FOUNDATION (BASE LAYER)

### GOAL
Create the core accounting structure required for all financial systems.

### TASKS

#### 1.1 Chart of Accounts (COA)
- Create `chart_of_accounts` table
- Fields:
  - id (uuid)
  - code
  - name
  - type (asset, liability, equity, revenue, expense)
  - subtype (bank, receivable, payable, payroll, etc.)
  - is_project_linkable (boolean)
  - is_owner_private (boolean)
  - parent_id (for hierarchy)
  - created_at

#### 1.2 General Ledger
- Create `gl_transactions` (header)
- Create `gl_entries` (lines)

Fields:
- gl_transactions:
  - id, date, reference, source_type, source_id, company_id

- gl_entries:
  - id, transaction_id, account_id
  - debit, credit
  - project_id (nullable)
  - description

#### 1.3 Core Rules
- Enforce double-entry (total debit = total credit)
- Prevent posting to inactive accounts
- Respect project-linkable rules

### GOAL

### TASKS
